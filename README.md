# 🛡️ Online Harassment Resources
A web-based resource platform designed to help individuals affected by online harassment, cyberbullying, social media abuse, blackmail, and other forms of cyber abuse. The website provides trusted reporting resources, safety guidance, privacy tools, legal information, and mental health support in one accessible location.

## 🎥 Project Demo
https://youtu.be/wEW9UhaUtsw

## ✨ Features
- Centralized online harassment support resources
- Platform-specific reporting guidance
- Government cybercrime reporting links
- Privacy and security recommendations
- Child protection resources
- Mental health support information
- Responsive and user-friendly interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🔒 Security Considerations
- Uses official reporting resources whenever possible
- Avoids unnecessary user input
- Keeps navigation simple and intuitive
- Promotes safe browsing by directing users to trusted platforms and government websites

## 🚀 Future Enhancements
- Search functionality
- Multi-language support
- Location-based emergency contacts
- FAQ section
- Accessibility improvements
- Additional international reporting resources
