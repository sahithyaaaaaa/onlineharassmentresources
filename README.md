### ONLINE HARASSMENT RESOURCES

### Video Demo: https://youtu.be/wEW9UhaUtsw

● About the Project:-
The purpose of this project is to create a simple and easy-to-use website that helps people who are experiencing online harassment. Many people, especially teenagers and young adults, face problems such as cyberbullying, blackmail, impersonation, stalking, sharing of private images, fake accounts, and abusive messages. Unfortunately, many victims do not know where to seek help or what actions they should take first.
This website was created to solve that problem by collecting trusted resources in one place. Instead of searching many different websites, users can find important reporting links, safety tips, child protection resources, legal information, government reporting portals, and mental health support on a single website.
The website is designed to be simple, responsive, and easy to understand. The interface avoids unnecessary complexity so that anyone can navigate through the resources without confusion.


● Project Motivation:-
The idea for this project came from the growing problem of online harassment around the world. Social media has become a major part of everyday life, but it has also created opportunities for cyberbullying, threats, impersonation, blackmail, and other forms of online abuse.
Many victims feel confused after experiencing harassment. They may not know whether to report the offender, contact the police, improve their account security, or seek emotional support. This project aims to guide users through these steps by organizing the available resources into different categories.
The main goal is not only to provide information but also to encourage victims to take action and understand that they are not alone.


● Website Structure:-
The website is divided into multiple pages that are connected together.

Home Page:
The homepage welcomes users with the message:

"Hey, you aren't alone!"

This message is meant to provide reassurance and remind users that support is available.

The homepage also contains a clickable "helplines" link. Clicking this link takes users to another page where they can choose the type of support they need.


● Helplines Page:-
The helplines page contains six clickable resource cards.

Each card represents a different category of support.

The categories are:
- Platform-Based Reporting
- Privacy and Security Tools
- Government Cybercrime Resources
- Law Enforcement Assistance
- Child Protection Services
- Mental Health Support

Each card is clickable and directs the user to a page containing detailed information about that topic.


● Information Pages:-
Every category page contains similar sections to make navigation easy.

Each page includes:
- A description explaining what the service is.
- Official reporting links.
- Information about evidence that should be collected.
- A simple reporting process.
- Helpful advice for victims.

The project uses official websites whenever possible to ensure that users are directed to trusted sources instead of unofficial websites.


● Technologies Used:-
This project was created using only the technologies taught during CS50x.

These include:
- HTML
- CSS
- JavaScript

-->HTML was used to build the structure of each webpage.
-->CSS was used to design the layout, spacing, typography, borders, and responsive appearance across different screen sizes.
-->JavaScript was included to improve user interaction and navigation where necessary.

No external frameworks were used because I wanted to build the project using the concepts that I learned during the course.


● Responsive Design:-
The website is designed to work on different screen sizes.

When viewed on a desktop or laptop, the pages make use of the available screen space while maintaining a clean layout.

When viewed on a mobile device, the layout automatically adjusts so that users can comfortably read the content without horizontal scrolling.

The six resource cards are displayed in a grid on larger screens and stack vertically on smaller screens for better accessibility.


● Security Considerations:-
Although this is an informational website, I still tried to follow good security practices while developing it.

Some of these include:
- Using only official reporting websites.
- Avoiding unnecessary user input.
- Avoiding external scripts that are not required.
- Using simple and clean HTML.
- Keeping navigation straightforward.
- Reducing the possibility of phishing by directing users only to trusted government and platform websites.

These practices make the website safer and more reliable for users.

● Challenges Faced:-
While building this project, I faced several challenges.

One challenge was making the website responsive across different devices while keeping the design simple.
Another challenge was organizing the information so that it remained easy to read without overwhelming the user.
I also spent time making sure that the resource cards, navigation, images, spacing, and page layouts looked consistent throughout the website.
Finding reliable official reporting resources for each category also required careful research.


● Future Improvements:-
If I continue developing this project, I would like to add more features such as:
- A search function for resources.
- Support for multiple languages.
- Location-based emergency contacts.
- Frequently Asked Questions (FAQ).
- Accessibility improvements for visually impaired users.
- Additional international reporting resources.

These improvements would make the website even more useful for a wider audience.


---> Final Thoughts:-
My goal was to create a website that is simple, welcoming, and informative. Instead of overwhelming users with large amounts of information, the website guides them step by step toward the appropriate resources.

I hope this project can help people understand that online harassment should never be ignored and that support is always available. Even small actions, such as reporting abusive content, improving account security, or reaching out for emotional support, can make a significant difference.

Overall, this project allowed me to combine programming with a meaningful social issue. It strengthened both my technical skills and my understanding of how technology can be used to create positive change.
